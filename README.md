# react-babylonjs-electron
Electron Starter Kit for 'react-babylonjs'

```bash
# Run in dev mode with HMR for content and dev tools open
yarn start-electron
```

```bash
# build windows portable
yarn package-win
# build mac
yarn package-mac
# build linux
yarn package-linux
```
